--- layout: page title: Tables description: The base style has minimal styling applied to it. It establishes preferred padding, font-size, and font-weight treatments. To enhance the look of the table, apply any additional classes listed below within the Modifiers section. ---

Modifiers

Bordered

When tables have multiple columns, it is suggested to add row borders to help the user more easily track the table data. This is suggested for data tables such as found within reports.

Hover Rows

Condensed Table

The condensed table does pretty much what it sounds like it would do: condenses the padding and line-height so that the table has a tighter feeling to it. This is suggested for data tables such as found within reports that have multiple records.

Text Alignment

Text alignment can be assigned at either the table-level or at the cell-level. In both cases, use the text-alignment atomic classes to assign how the cell(s) should align text: .ta-left, .ta-center, .ta-right, or .ta-justify.

Unique Layouts

These are different layouts which can be added to particular areas within a table.

Totals Row

Used mainly with data tables, the totals row increases the font-size for all cells within a row.
